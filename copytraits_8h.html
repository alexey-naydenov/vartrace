Detailed Description

Describe copy mechanism for different types.

Vartrace::Log() template function calls appropriate overloaded Vartrace::DoLog() function that does actual copying. The kind of logging function is chosen based on type trait. If the template structure #CopyTraits is not specialized for a type then objects of that type are copied through memory copy. Logging of simple types are sped up by using assignment.
